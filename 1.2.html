Un conocimiento muy util sin lugar a duda es el de poder cambiar tu sistema operativo sin problema.

Por tanto veremos como instalar 2 sistemas lideres en el mercado :

Windows 10

Las caracteristicas mas importantes de este sistema son :

  • El mas pupular a nivel mundial
  • Abundancia de software libre y de pago
  • Software de categoria mundial (adobe, autocad, diseño, etc...)
  • Estable
  • Asistente inteligente (cortana)
  • Bajos Requisitos en generacion actual de pc:
    • 2 GHz compatible con PAE, NX y SSE2
    • 1 GB (32 bits) o 2 GB (64 bits)
    • 16 GB (32 bits) o 20 GB (64 bits) HDD
    • Dispositivo gráfico Microsoft DirectX 9 con controlador WDDM
  • Paqueteria Office
  • Buen Rendimiento
  • Ideal para Gamers

Instalacion

Para la instalacion requrimos lo siguiente :

  • USB 8 GB
  • Media Creation tool (proporcionado por microsoft)
  • Respaldo de Informacion personal

La usb perdera todo su contenido

Conseguir el ISO y preparar la USB

Microsoft resuelve el problema por nosotros con su herramienta media creation tool. con la usb insertada vamos a google y buscamos : media creation tool. Nos dirigimos a el primer resultado de la pagina oficial de microsoft microsoft.com

Resultado de la busqueda Solo basta con dar click sobre el boton para descargar Almacenamos el archivo una vez descargado lo ejecutamos dejamos que analize nuestro sistema actual Leemos y aceptamos los terminos Esperamos un poco Le indicamos que queremos una usb de instalacion quitamos el checkbox para que descarge los ultimos archivos, solo seleccionamos nuestro idioma Le indicamos que sera un medio de instalacion USB Aqui deberiamos ver nuestra usb, la seleccionamos y damos a siguiente Esperamos a que descarge los archivos Una vez descargado se empezara a crear el medio, no retirar la usb Listo Nuestra USB esta lista

Instalacion

La instalacion de el sistema es muy sencilla basta con seguir las intrucciones de el instalador, solo hay un detalle. Al llegar a la parte donde se selecciona el disco para su instalacion debes borrar todas las particiones una a una hasta que solo te quede espacio sin asignar en el disco duro.

Se puede hacer de forma mas rapida con diskpart pero require un poco mas de conocimiento en el uso de el CMD. Dejo link advirtiendo que solo es util la limpieza dado que el particionado lo hace el instalador de windows en automatico
